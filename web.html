What I Covered In This Module

In this module, I have covered the fundamentals of developing a responsive and mobile-first front end for a web application. This includes understanding web design concepts, web development tools and frameworks used in industry, HTML, CSS and proficiency in the JavaScript scripting language. I have also developed skills in application testing and web optimisation to provide me with skills to develop website solutions that are effective, responsive, perform well and comply with industry standards.

Examples of Work

During the first assignment of this module, I was tasked with creating a presentation that evaluates a businesses current web solution and then develop effective improvements based upon the current performance and usability. I was also tasked with researching and evaluating potential choices of tools and frameworks that could have been used to implement these changes. For example, Bootstrap for quick and efficient development and GitHub for version control. For the second assignment of this module, I was then tasked with designing, developing and testing a responsive and mobile-first front end for a web application that improves upon the website analysed in assignment 1.
